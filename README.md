Multichain DApp

Веб-приложение для взаимодействия с кошельком MetaMask, отправки транзакций и получения информации о токенах. Проект выполнен на базе Next.js, TypeScript и Viem.

Функциональность

Подключение кошелька

Определение активной сети.

Получение адреса пользователя.

Отображение баланса.

Отправка транзакций

Ввод суммы в GWei.

Указание адреса получателя.

Отправка транзакции через MetaMask.

Работа с токенами (ERC-20)

Ввод адреса смарт-контракта.

Загрузка ABI.

Чтение имени токена, символа, баланса и количества десятичных знаков.
