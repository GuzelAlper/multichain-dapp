Multichain DApp
Веб-приложение для взаимодействия с кошельком MetaMask, выполнения транзакций и получения информации о токенах. Проект реализован с использованием Next.js, TypeScript и библиотеки Viem.

Функциональность:

Подключение кошелька

Определение активной сети

Получение адреса пользователя

Отображение баланса

Отправка транзакций

Ввод суммы в GWei

Указание адреса получателя

Отправка транзакции через MetaMask

Работа с токенами ERC-20

Ввод адреса смарт-контракта

Загрузка ABI

Получение названия токена, символа, баланса и количества десятичных знаков

Технологии:
Next.js
TypeScript
Viem
MetaMask
React Hooks

Установка и запуск:

Установка зависимостей:
npm install

Запуск в режиме разработки:
npm run dev

Требования:
Установленный MetaMask и выбранная тестовая сеть (например, Sepolia).

Структура проекта:

app/
client.ts
walletComponent.tsx
tokenComponent.tsx
transactionComponent.tsx
public/
abi.ts
next.config.ts
package.json
tsconfig.json

Примечание:
Каталоги .next и node_modules не включаются в репозиторий и создаются автоматически при сборке и установке зависимостей.
